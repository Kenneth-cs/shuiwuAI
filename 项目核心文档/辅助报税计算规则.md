# 税悟 AI - 辅助报税页面计算规则文档

本文档详细说明辅助报税页面中所有填写项的计算逻辑和规则。

---

## 一、增值税及附加税费申报表（小规模纳税人适用）

### 基础信息
- **纳税人识别号**：从 AI 提取的公司信息中获取
- **纳税人名称**：从 AI 提取的公司信息中获取
- **税款所属期**：从 AI 提取的期间信息中获取（格式：YYYY-MM）

### 计税依据（栏次 1-14）

#### 1、应征增值税不含税销售额（3%征收率）
**栏次**：1（货物及劳务）、1（服务、不动产和无形资产）
**计算规则**：
- 货物及劳务：取原始数据中的 `taxable_3_percent` 中属于货物部分（目前默认 0）
- 服务不动产：取原始数据中的 `taxable_3_percent`
**数据来源**：AI 提取的 `revenue.taxable_3_percent`

#### 2、增值税专用发票不含税销售额
**栏次**：2
**计算规则**：
- 需用户手动填写或从发票数据中提取
- 目前默认 0（假设小规模纳税人不开具专票或数据未采集）

#### 3、其他增值税发票不含税销售额
**栏次**：3
**计算规则**：
- 货物及劳务：0
- 服务不动产：取原始数据中的 `taxable_1_percent`（1%征收率的应税销售额）
**数据来源**：AI 提取的 `revenue.taxable_1_percent`

#### 4、应征增值税不含税销售额（5%征收率）
**栏次**：4
**计算规则**：
- 适用于不动产租赁等特殊业务
- 目前默认 0（假设无 5%征收率业务）
**注意**：货物及劳务栏显示 `--`（不适用）

#### 5-6、5%征收率的专票/其他发票销售额
**栏次**：5、6
**计算规则**：默认 0

#### 7、销售使用过的固定资产不含税销售额
**栏次**：7
**计算规则**：
- 货物及劳务：默认 0
- 服务不动产：显示 `--`（不适用）
**政策说明**：销售使用过的固定资产可按 3%征收率减按 2%征收

#### 8、其中：其他增值税发票不含税销售额
**栏次**：8
**计算规则**：默认 0（栏次 7 的明细项）

#### 9、免税销售额
**栏次**：9
**计算规则**：
- 货物及劳务：0
- 服务不动产：取原始数据中的 `tax_free`
**数据来源**：AI 提取的 `revenue.tax_free`
**政策说明**：月销售额 ≤ 10 万元（季度 30 万）可享受免税

#### 10、其中：小微企业免税销售额
**栏次**：10
**计算规则**：
- 当满足免税条件时（月销售额 ≤ 10 万），等于栏次 9 的金额
- 否则为 0
**判定逻辑**：`totalSales <= 100000`

#### 11、未达起征点销售额
**栏次**：11
**计算规则**：默认 0（适用于个体工商户等）

#### 12、其他免税销售额
**栏次**：12
**计算规则**：
- 当不满足小微企业免税条件但有其他免税政策时填写
- 目前默认 0

#### 13、出口免税销售额
**栏次**：13
**计算规则**：默认 0（假设无出口业务）

#### 14、其中：其他增值税发票不含税销售额
**栏次**：14
**计算规则**：默认 0（栏次 13 的明细项）

### 税款计算（栏次 15-22）

#### 15、本期应纳税额
**栏次**：15
**计算规则**：
```
应征增值税不含税销售额 × 适用征收率
= (栏次 1 × 3%) + (栏次 3 × 1%)
= (taxable_3_percent × 0.03) + (taxable_1_percent × 0.01)
```
**特殊处理**：当满足免税条件时（月销售额 ≤ 10 万），本期应纳税额 = 0

#### 16、本期应纳税额减征额
**栏次**：16
**计算规则**：
- 适用于销售使用过的固定资产减征（3%减按 2%，减征 1%）
- 目前默认 0

#### 17、本期免税额
**栏次**：17
**计算规则**：
- 当满足免税条件时 = 栏次 15 的应纳税额（即免掉的税额）
- 否则 = 0

#### 18、其中：小微企业免税额
**栏次**：18
**计算规则**：
- 当满足小微企业免税条件时 = 栏次 17 的金额
- 否则 = 0

#### 19、未达起征点免税额
**栏次**：19
**计算规则**：默认 0

#### 20、应纳税额合计
**栏次**：20
**计算公式**：`栏次 15 - 栏次 16`
**逻辑**：
```
if (isVatExempt) 应纳税额合计 = 0
else 应纳税额合计 = 本期应纳税额 - 本期应纳税额减征额
```

#### 21、本期预缴税额
**栏次**：21
**计算规则**：默认 0（本期已预缴的税款）
**注意**：本年累计列显示 `--`（不参与累计）

#### 22、本期应补（退）税额
**栏次**：22
**计算公式**：`栏次 20 - 栏次 21`
**本年累计**：显示 `--`（按税款所属期汇总，不按季度累计）

### 附加税费（栏次 23-25）

**政策依据**：财税〔2019〕13 号，小规模纳税人"六税两费"减征 50%

#### 23、城市维护建设税本期应补（退）税额
**栏次**：23
**计算步骤**：
```
1. 计算城建税基数 = 栏次 22（本期应补退增值税额）
2. 城建税全额 = 基数 × 7%（市区）
3. 减征额 = 城建税全额 × 50%
4. 本期应补（退）税额 = 城建税全额 - 减征额
```
**示例**：
- 增值税应纳税额 297.03 元
- 城建税全额 = 297.03 × 7% = 20.79 元
- 减征额 = 20.79 × 50% = 10.40 元
- 应缴城建税 = 20.79 - 10.40 = 10.39 元

#### 24、教育费附加本期应补（退）费额
**栏次**：24
**计算步骤**：
```
1. 教育费附加全额 = 基数 × 3%
2. 减征额 = 全额 × 50%
3. 应缴费额 = 全额 - 减征额
```
**示例**：
- 教育费附加全额 = 297.03 × 3% = 8.91 元
- 减征额 = 8.91 × 50% = 4.46 元
- 应缴教育费附加 = 8.91 - 4.46 = 4.45 元

#### 25、地方教育附加本期应补（退）费额
**栏次**：25
**计算步骤**：
```
1. 地方教育附加全额 = 基数 × 2%
2. 减征额 = 全额 × 50%
3. 应缴费额 = 全额 - 减征额
```
**示例**：
- 地方教育附加全额 = 297.03 × 2% = 5.94 元
- 减征额 = 5.94 × 50% = 2.97 元
- 应缴地方教育附加 = 5.94 - 2.97 = 2.97 元

## 二、企业所得税月（季）度预缴纳税申报表（A类）

### 基础信息
- **税款所属期间**：从 AI 提取的期间信息
- **纳税人识别号/名称**：从 AI 提取的公司信息

### 优惠及附报事项

#### 从业人数（季度平均值）
**数据来源**：`company_status.employee_count`
**说明**：填报季度内各月平均从业人数

#### 资产总额（万元）
**数据来源**：`company_status.asset_total_myr`
**单位**：万元
**说明**：填报季度末资产总额，用于判断小型微利企业

#### 国家限制或禁止行业
**默认值**：☑ 否
**说明**：如属于限制/禁止行业，不能享受小微优惠

#### 小型微利企业判定
**判定逻辑**（需同时满足）：
```
isSmallMicro = (
  employee_count <= 300 
  && asset_total_myr <= 5000 
  && realProfit <= 3000000
)
```
**结果**：
- ☑ 是：适用 5% 实际税率（25% × 20%）
- ☐ 否：适用 25% 税率

### 预缴税款计算（行次 1-32）

#### 1、营业收入
**行次**：1
**数据来源**：`revenue.total`
**说明**：本期含税收入总额

#### 1.1-1.3、出口收入明细
**行次**：1.1、1.2、1.3
**计算规则**：默认 0（假设无出口业务）

#### 2、营业成本
**行次**：2
**数据来源**：`profit_info.total_cost` 或 `revenue.total - profit_info.total_profit`
**说明**：本期营业总成本

#### 3、税金及附加
**行次**：3
**计算规则**：`城建税 + 教育费附加 + 地方教育附加`
**数据来源**：增值税计算结果中的附加税费合计

#### 4、销售费用
**行次**：4
**计算规则**：默认 0（需从财务报表提取）

#### 5、管理费用
**行次**：5
**计算规则**：默认 0（需从财务报表提取）

#### 6、研发费用
**行次**：6
**计算规则**：默认 0（可享受加计扣除，但预缴时不扣减）

#### 7、财务费用
**行次**：7
**计算规则**：默认 0（需从财务报表提取）

#### 8、其他收益
**行次**：8
**计算规则**：默认 0（政府补助等）

#### 9、投资收益
**行次**：9
**计算规则**：默认 0
**子项**：9.1 股权投资处置收益、9.2 其他投资收益

#### 10-14、其他损益项目
**行次**：10-14
**包括**：净敞口套期收益、公允价值变动收益、信用减值损失、资产减值损失、资产处置收益
**计算规则**：默认 0

#### 15、营业利润
**行次**：15
**计算公式**：
```
营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用
+ 其他收益 + 投资收益 + 净敞口套期收益 + 公允价值变动收益
+ 信用减值损失 + 资产减值损失 + 资产处置收益
```
**简化计算**：目前等于利润总额（假设无营业外收支）

#### 16、营业外收入
**行次**：16
**计算规则**：默认 0

#### 17、营业外支出
**行次**：17
**计算规则**：默认 0

#### 18、利润总额
**行次**：18
**计算公式**：`行次 15 + 行次 16 - 行次 17`
**数据来源**：`profit_info.total_profit`

#### 19、特定业务计算的应纳税所得额
**行次**：19
**计算规则**：默认 0（适用于房地产企业预售等特定业务）
**子项**：19.1 销售未完工产品的收入

#### 20、不征税收入
**行次**：20
**计算规则**：默认 0（如财政拨款等）

#### 21、资产加速折旧、摊销（扣除）调减额
**行次**：21
**计算规则**：默认 0（需填写 A201020 附表）

#### 22、免税收入、减计收入、加计扣除
**行次**：22
**计算规则**：默认 0（如国债利息收入等）

#### 23、所得减免
**行次**：23
**计算规则**：默认 0（如农林牧渔项目所得减免）

#### 24、弥补以前年度亏损
**行次**：24
**计算规则**：默认 0（不超过 5 年内的亏损）

#### 25、实际利润额
**行次**：25
**计算公式**：
```
行次 18（利润总额）+ 行次 19 - 行次 20 - 行次 21 - 行次 22 - 行次 23 - 行次 24
```
**简化计算**：目前等于利润总额（假设其他调整项为 0）
**注意**：如为负数，填 0

#### 26、税率
**行次**：26
**固定值**：25%
**说明**：法定税率，无论是否小微均填 25%

#### 27、应纳所得税额
**行次**：27
**计算公式**：`行次 25 × 行次 26（25%）`
**说明**：按法定税率计算的税额，非实际缴纳额

#### 28、减免所得税额
**行次**：28
**计算公式**：
```
if (isSmallMicro) {
  减免额 = 实际利润额 × (25% - 5%) = 实际利润额 × 20%
} else {
  减免额 = 其他优惠政策减免额（如高新技术企业 15% 税率差额）
}
```
**子项**：28.1 符合条件的小型微利企业减免企业所得税

#### 29、抵免所得税额
**行次**：29
**计算规则**：默认 0（如购买环保设备抵免）

#### 30、本年累计已预缴所得税额
**行次**：30
**计算规则**：默认 0（本年内已预缴的税款）

#### 31、特定业务预缴（征）所得税额
**行次**：31
**计算规则**：默认 0

#### 32、本期应补（退）所得税额
**行次**：32
**计算公式**：
```
行次 27 - 行次 28 - 行次 29 - 行次 30 - 行次 31
```
**简化计算**：
```
if (isSmallMicro) {
  应缴税额 = 实际利润额 × 5%  // 小微优惠后实际税率
} else {
  应缴税额 = 实际利润额 × 25%
}
```

#### 39、民族自治地区企业所得税地方分享部分
**行次**：39
**计算规则**：默认 0（勾选"无"）
**说明**：民族自治地区可减免地方分享的 40% 部分

#### 40、实际应补（退）所得税额
**行次**：40
**计算公式**：等于行次 32
**说明**：最终实际缴纳金额

## 三、文化事业建设费申报表

### 计费收入（栏次 1-2）

#### 1、应征收入
**栏次**：1
**计算规则**：广告/娱乐服务的应税销售额
**数据来源**：增值税计算中的 `totalTaxableSales`
**条件**：仅当 `industry_type` 为 'ad' 或 'entertainment' 时征收

#### 2、免征收入
**栏次**：2
**计算规则**：符合免征条件的收入
**目前**：默认 0

### 费额计算（栏次 3-9）

#### 3、减除项目期初金额
**栏次**：3
**计算规则**：上期结转的减除项目余额
**目前**：默认 0
**本年累计**：显示 `--`（不适用）

#### 4、减除项目本期发生额
**栏次**：4
**计算规则**：本期新增的允许减除的金额（如广告发布成本）
**目前**：默认 0

#### 5、应征收入减除额
**栏次**：5
**计算规则**：本期实际减除的应征收入部分
**目前**：默认 0

#### 6、免征收入减除额
**栏次**：6
**计算规则**：本期实际减除的免征收入部分
**目前**：默认 0

#### 7、减除项目期末余额
**栏次**：7
**计算公式**：`栏次 3 + 栏次 4 - 栏次 5 - 栏次 6`
**说明**：结转至下期的减除项目余额

#### 8、计费销售额
**栏次**：8
**计算公式**：`栏次 1 - 栏次 5`
**说明**：应征收入减去减除额后的实际计费基数

#### 9、费率
**栏次**：9
**固定值**：3%
**说明**：文化事业建设费费率

### 费额缴纳（栏次 10-19）

#### 10、应缴费额
**栏次**：10
**计算公式**：`栏次 8 × 栏次 9（3%）`
**示例**：计费销售额 50,000 × 3% = 1,500 元

#### 11、期初未缴费额
**栏次**：11
**计算规则**：上期欠缴或多缴的费额（多缴为负数）
**目前**：默认 0
**本年累计**：显示 `--`

#### 12、本期已缴费额
**栏次**：12
**计算公式**：`栏次 13 + 栏次 14 + 栏次 15`
**说明**：本期实际已缴纳的各项费额合计

#### 13-15、已缴费额明细
**行次**：13（本期预缴）、14（缴纳上期）、15（缴纳欠费）
**计算规则**：默认 0

#### 16、期末未缴费额
**栏次**：16
**计算公式**：`栏次 10 + 栏次 11 - 栏次 12 - 财税减征额`
**说明**：本期末欠缴或多缴的费额
**目前**：默认等于应缴费额（假设未缴纳）

#### 17、其中：欠缴费额
**栏次**：17
**计算公式**：`栏次 11 - 栏次 14 - 栏次 15`
**说明**：纯欠缴金额（不含本期应缴）
**本年累计**：显示 `--`

#### 18、本期应补（退）费额
**栏次**：18
**计算公式**：`栏次 10 × (1 - 减征比例) - 栏次 13`
**说明**：本期实际应补缴的费额
**目前**：默认等于应缴费额（假设无减征、无预缴）
**本年累计**：显示 `--`

#### 19、本期检查已补缴费额
**栏次**：19
**计算规则**：税务检查后补缴的费额
**目前**：默认 0

## 四、财务报表（小企业会计准则）

### 利润表（会小企 02 表）

#### 1、营业收入
**行次**：1
**数据来源**：`revenue.total`
**说明**：本期含税收入总额

#### 2、营业成本
**行次**：2
**数据来源**：`profit_info.total_cost` 或 `revenue.total - profit_info.total_profit`

#### 3、税金及附加
**行次**：3
**计算规则**：增值税计算结果中的附加税费合计
**包括**：城建税、教育费附加、地方教育附加

#### 3.1-3.x、税金及附加明细
**包括**：消费税、营业税、城建税、资源税、土地增值税、房产税等
**目前**：默认显示城建税和教育费附加，其余为 0

#### 11、销售费用
**行次**：11
**计算规则**：默认 0

#### 14、管理费用
**行次**：14
**计算规则**：默认 0
**子项**：研究费用（可加计扣除）

#### 18、财务费用
**行次**：18
**计算规则**：默认 0
**子项**：利息费用、利息收入、汇兑损失

#### 22、营业利润
**行次**：22
**计算公式**：
```
营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
```

#### 23、营业外收入
**行次**：23
**计算规则**：默认 0
**子项**：政府补助

#### 25、营业外支出
**行次**：25
**计算规则**：默认 0
**子项**：坏账损失、税收滞纳金等

#### 31、利润总额
**行次**：31
**计算公式**：`营业利润 + 营业外收入 - 营业外支出`
**数据来源**：`profit_info.total_profit`

#### 32、所得税费用
**行次**：32
**计算规则**：企业所得税计算结果中的 `taxDue`

#### 33、净利润
**行次**：33
**计算公式**：`利润总额 - 所得税费用`

```javascript
function calculateTaxes(data) {
  // 基础数据
  const rev = data.revenue;
  const profit = data.profit_info;
  const status = data.company_status;
  
  // 1. 增值税计算
  const vat1pct = rev.taxable_1_percent * 0.01;
  const vat3pct = (rev.taxable_3_percent || 0) * 0.03;
  const totalTaxableSales = rev.taxable_1_percent + (rev.taxable_3_percent || 0);
  const totalSales = totalTaxableSales + (rev.tax_free || 0);
  const isVatExempt = totalSales <= 100000;  // 免税判定
  const vatDue = isVatExempt ? 0 : (vat1pct + vat3pct);
  
  // 2. 附加税费（六税两费减征 50%）
  const urbanBuild = vatDue * 0.07;      // 城建税 7%
  const eduSurcharge = vatDue * 0.03;    // 教育费附加 3%
  const localEdu = vatDue * 0.02;        // 地方教育附加 2%
  const finalUrban = urbanBuild * 0.5;   // 减征后
  const finalEdu = eduSurcharge * 0.5;
  const finalLocal = localEdu * 0.5;
  
  // 3. 企业所得税
  const realProfit = profit.total_profit;
  const isSmallMicro = status.employee_count <= 300 
                      && status.asset_total_myr <= 5000 
                      && realProfit <= 3000000;
  const actualCitRate = isSmallMicro ? 0.05 : 0.25;  // 小微 5%，一般 25%
  const citDue = realProfit > 0 ? realProfit * actualCitRate : 0;
  const citReduction = isSmallMicro && realProfit > 0 
                      ? realProfit * (0.25 - actualCitRate) : 0;
  
  // 4. 文化事业建设费
  const isAdOrEntertainment = ['ad', 'entertainment'].includes(status.industry_type);
  const cultureDue = isAdOrEntertainment ? totalTaxableSales * 0.03 : 0;
  
  return { /* 各税种计算结果 */ };
}
```

## 八、v2.0 新增计算项详细说明

### 增值税新增计算项

#### 5%征收率（不动产租赁）
**栏次**：4
**适用场景**：
- 小规模纳税人出租不动产（不含住房）
- 销售2016年4月30日前取得的不动产
- 劳务派遣服务选择差额纳税

**计算规则**：
```
应纳税额 = taxable_5_percent × 5%
```
**数据来源**：`revenue.taxable_5_percent`

#### 销售使用过的固定资产减征
**栏次**：7、8、16
**政策依据**：财税〔2009〕9号
**计算规则**：
```
销售额 = used_fixed_assets
全额税额 = 销售额 × 3%
减征额 = 销售额 × 1%（即减征50%的税额）
实缴税额 = 销售额 × 2%（3% - 1%）
```
**填报方式**：
- 栏次7：销售额
- 栏次16：减征额（销售额×1%）
- 实际按2%征收

#### 出口退（免）税
**栏次**：13、14
**计算规则**：
```
if (has_export_license) {
  出口销售额免税，税额 = 0
} else {
  出口销售额按征收率征税
}
```
**数据来源**：
- `revenue.export_sales`：出口销售额
- `revenue.has_export_license`：是否有出口资质

### 企业所得税新增计算项

#### 特定业务（房地产预售）
**行次**：19、19.1
**适用场景**：房地产开发企业预售未完工产品
**计算规则**：
```
特定业务应纳税所得额 = specific_business_income × 预计毛利率
```
**预计毛利率**（一般情况）：
- 开发项目位于省、自治区、直辖市和计划单列市人民政府所在地城市城区和郊区的，不得低于15%
- 位于地级市城区及郊区的，不得低于10%
- 位于其他地区的，不得低于5%

**数据来源**：`cit_adjustments.specific_business_income`

#### 资产加速折旧、摊销（扣除）调减额
**行次**：21
**适用场景**：
- 固定资产加速折旧（技术进步、更新换代快）
- 一次性扣除（单位价值不超过500万元的设备器具）

**计算规则**：
```
调减额 = 加速折旧额 - 税法允许扣除额（直线法）
或
调减额 = 一次性扣除全额（500万以下设备）
```
**数据来源**：`cit_adjustments.asset_acceleration_deduction`
**附表**：需填写 A201020《资产加速折旧、摊销(扣除)优惠明细表》

#### 研发费用加计扣除
**行次**：22（包含在"免税收入、减计收入、加计扣除"中）
**政策依据**：财税〔2023〕7号（2023年起100%加计扣除）
**计算规则**：
```
研发费用 = rd_expense
加计扣除额 = 研发费用 × 100%
应纳税所得额调减 = 加计扣除额
```
**数据来源**：
- `profit_info.rd_expense`：研发费用金额
- 计算得出 `cit.rdExtraDeduction = rd_expense × 100%`

**研发活动范围**：
- 新技术、新产品、新工艺的研发
- 实质性改进技术、产品、工艺
- 不包括：常规性升级、直接应用科研成果、商品化后技术支持活动

#### 免税收入
**行次**：20（不征税收入）、22（免税收入）
**免税收入范围**：
1. 国债利息收入
2. 符合条件的居民企业之间的股息、红利等权益性投资收益
3. 在中国境内设立机构、场所的非居民企业从居民企业取得与该机构、场所有实际联系的股息、红利等权益性投资收益
4. 符合条件的非营利组织的收入

**计算规则**：
```
应纳税所得额调减 = tax_exempt_income
```
**数据来源**：`cit_adjustments.tax_exempt_income`

#### 弥补以前年度亏损
**行次**：24
**政策依据**：企业所得税法第十八条
**计算规则**：
```
可弥补亏损 = 以前年度亏损（不超过5年）
应纳税所得额调减 = min(可弥补亏损, 当年应纳税所得额)
```
**数据来源**：`cit_adjustments.loss_compensate`

#### 高新技术企业优惠
**判定条件**：`company_status.is_high_tech = true`
**优惠税率**：15%（法定税率25%）
**计算规则**：
```
if (isHighTech) {
  实际税率 = 15%
  减免税额 = 应纳税所得额 × (25% - 15%) = 应纳税所得额 × 10%
}
```
**优先级**：高新技术企业优惠与小微企业优惠不叠加，取最优惠者

### 文化事业建设费新增计算项

#### 减除项目明细
**栏次**：3-7
**适用场景**：广告代理业务，可扣除支付给其他广告公司或广告发布者的广告发布费

**计算规则**：
```
期初余额 = culture_deductions.opening_balance
本期发生额 = culture_deductions.current_occurrence
本期实际减除额 = culture_deductions.actual_deduction
期末余额 = 期初余额 + 本期发生额 - 本期实际减除额
计费销售额 = 应征收入 - 本期实际减除额
应缴费额 = 计费销售额 × 3%
```

**约束条件**：
- 减除额不得超过应征收入
- 需取得合法有效的减除凭证

## 九、v2.0 核心计算引擎代码（完整版）

```javascript
function calculateTaxes(data) {
  const rev = data.revenue;
  const profit = data.profit_info;
  const status = data.company_status;
  const adj = data.cit_adjustments || {};
  const culDed = data.culture_deductions || {};

  // ========== 1. 增值税（完整版）==========
  const vat1pct = (rev.taxable_1_percent || 0) * 0.01;
  const vat3pct = (rev.taxable_3_percent || 0) * 0.03;
  const vat5pct = (rev.taxable_5_percent || 0) * 0.05;          // 【v2.0】5%征收率
  const usedAssetSales = rev.used_fixed_assets || 0;
  const usedAssetVatDue = usedAssetSales * 0.02;                   // 【v2.0】3%减按2%
  const usedAssetReduction = usedAssetSales * 0.01;                // 【v2.0】减征1%
  const exportSales = rev.export_sales || 0;
  const exportVat = rev.has_export_license ? 0 : exportSales * 0.01; // 【v2.0】出口

  const totalTaxableSales = (rev.taxable_1_percent || 0) + (rev.taxable_3_percent || 0) +
                            (rev.taxable_5_percent || 0) + usedAssetSales + exportSales;
  const totalSales = totalTaxableSales + (rev.tax_free || 0);
  const isVatExempt = totalSales <= 100000;
  const vatDue = isVatExempt ? 0 : (vat1pct + vat3pct + vat5pct + usedAssetVatDue + exportVat);

  // ========== 2. 附加税费 ==========
  const urbanBuild = (vatDue * 0.07) * 0.5;    // 减征50%
  const eduSurcharge = (vatDue * 0.03) * 0.5;
  const localEdu = (vatDue * 0.02) * 0.5;

  // ========== 3. 企业所得税（完整版）==========
  const realProfit = profit.total_profit;

  // 【v2.0】费用明细
  const salesExpense = profit.sales_expense || 0;
  const manageExpense = profit.manage_expense || 0;
  const rdExpense = profit.rd_expense || 0;
  const financeExpense = profit.finance_expense || 0;

  // 【v2.0】营业外收支
  const nonbusinessIncome = profit.nonbusiness_income || 0;
  const nonbusinessExpense = profit.nonbusiness_expense || 0;

  // 【v2.0】调整项
  const specificBusiness = adj.specific_business_income || 0;
  const taxExemptIncome = adj.tax_exempt_income || 0;
  const rdExtraDeduction = rdExpense * 1.0;                     // 100%加计
  const assetAccDeduction = adj.asset_acceleration_deduction || 0;
  const lossCompensate = adj.loss_compensate || 0;

  // 【v2.0】应纳税所得额计算
  let taxableIncome = realProfit;
  taxableIncome += specificBusiness;
  taxableIncome -= taxExemptIncome;
  taxableIncome -= assetAccDeduction;
  taxableIncome -= rdExtraDeduction;
  taxableIncome -= lossCompensate;
  if (taxableIncome < 0) taxableIncome = 0;

  // 税率判定
  const isSmallMicro = (status.employee_count || 0) <= 300 &&
                       (status.asset_total_myr || 0) <= 5000 &&
                       realProfit <= 3000000;
  const isHighTech = status.is_high_tech || false;

  let actualCitRate = 0.25;
  if (isSmallMicro) actualCitRate = 0.05;
  else if (isHighTech) actualCitRate = 0.15;

  const citDue = taxableIncome * actualCitRate;
  const citReduction = (taxableIncome * 0.25) - citDue;

  // ========== 4. 文化事业建设费（完整版）==========
  const isAdOrEntertainment = ['ad', 'entertainment'].includes(status.industry_type);
  const dedOpening = culDed.opening_balance || 0;
  const dedOccurrence = culDed.current_occurrence || 0;
  const dedActual = culDed.actual_deduction || 0;
  const dedClosing = dedOpening + dedOccurrence - dedActual;
  const cultureTaxableSales = Math.max(0, (isAdOrEntertainment ? totalTaxableSales : 0) - dedActual);
  const cultureDue = cultureTaxableSales * 0.03;

  // ========== 5. 财务报表 ==========
  const operatingProfit = (rev.total || 0) - (profit.total_cost || 0) - (vatDue * 0.12) -
                          salesExpense - manageExpense - financeExpense;
  const profitBeforeTax = operatingProfit + nonbusinessIncome - nonbusinessExpense;
  const netProfit = profitBeforeTax - citDue;

  return {
    // 增值税
    vat: {
      vatDue, vatDue1pct: r(vat1pct), vatDue3pct: r(vat3pct),
      vatDue5pct: r(vat5pct),                                    // 【v2.0】
      vatDueUsedAsset: r(usedAssetVatDue),                       // 【v2.0】
      usedAssetReduction: r(usedAssetReduction),                 // 【v2.0】
      vatDueExport: r(exportVat),                                // 【v2.0】
      finalUrban: r(urbanBuild), finalEdu: r(eduSurcharge), finalLocal: r(localEdu),
    },
    // 企业所得税
    cit: {
      revenue: r(rev.total), totalCost: r(profit.total_cost || 0),
      salesExpense: r(salesExpense), manageExpense: r(manageExpense),  // 【v2.0】
      rdExpense: r(rdExpense), financeExpense: r(financeExpense),      // 【v2.0】
      operatingProfit: r(operatingProfit),                             // 【v2.0】
      nonbusinessIncome: r(nonbusinessIncome),                         // 【v2.0】
      nonbusinessExpense: r(nonbusinessExpense),                       // 【v2.0】
      specificBusiness: r(specificBusiness),                           // 【v2.0】
      taxExemptIncome: r(taxExemptIncome),                             // 【v2.0】
      rdExtraDeduction: r(rdExtraDeduction),                           // 【v2.0】
      assetAccDeduction: r(assetAccDeduction),                         // 【v2.0】
      lossCompensate: r(lossCompensate),                               // 【v2.0】
      realProfit: r(profitBeforeTax),
      taxableIncome: r(taxableIncome),                                 // 【v2.0】
      taxDue: r(citDue), reduction: r(citReduction),
      isSmallMicro, isHighTech,
      netProfit: r(netProfit),                                          // 【v2.0】
    },
    // 文化事业建设费
    culture: {
      dedOpening: r(dedOpening), dedOccurrence: r(dedOccurrence),       // 【v2.0】
      dedActual: r(dedActual), dedClosing: r(dedClosing),              // 【v2.0】
      taxableSales: r(cultureTaxableSales), due: r(cultureDue),
    },
  };
}

function r(v) { return Math.round((v || 0) * 100) / 100; }
```
